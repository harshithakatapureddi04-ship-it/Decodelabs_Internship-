# 📚 Online Book Store Website

## Project Overview

This project is a responsive Online Book Store Website developed using HTML, CSS, and JavaScript. The website provides a simple and interactive platform for users to browse books, register, log in, and contact the bookstore.

## Features

### Home Page

* Welcome section
* Attractive bookstore image
* Interactive "Explore Books" button
* Dark Mode toggle

### Register Page

* User registration form
* Password strength checker
* Registration success message
* Responsive design

### Login Page

* User login form
* Show/Hide password feature
* Login success message
* Dark Mode toggle

### Books Page

* Display of available books
* Book search functionality
* Add to Cart feature
* Today's Offer button
* Responsive book cards with hover effects

### Contact Page

* Contact form
* Character counter for messages
* Success confirmation message
* Responsive layout

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Responsive Web Design

## How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `home.html` in a browser or use the Live Server extension.
4. Navigate through the pages using the navigation menu.
